# 💳 Bank Transaction Data Analysis (Excel & Power BI)

## 📌 Project Overview

This project focuses on analyzing bank transaction data to identify spending patterns, transaction trends, and financial insights. The analysis is performed using Microsoft Excel for data cleaning and Power BI for interactive dashboard visualization.

The objective is to transform raw transaction data into meaningful insights that can help in financial decision-making and fraud detection.

---

## 🎯 Objectives

* Analyze customer transaction behavior
* Identify high-value transactions
* Track spending patterns over time
* Detect unusual or suspicious transactions

---

## 🛠️ Tools & Technologies

* Microsoft Excel (Data Cleaning & Preparation)
* Power BI (Dashboard & Visualization)
* Data Analysis Techniques

---

## 📁 Dataset Information

* **Type:** Bank Transaction Data
* **Format:** Excel (.xlsx)
* **Key Fields:**

  * Transaction ID
  * Date
  * Amount
  * Transaction Type (Credit/Debit)
  * Account Details
  * Balance

---

## 🔍 Data Cleaning Process (Excel)

* Removed null/missing values
* Standardized date formats
* Removed duplicate transactions
* Converted columns to proper data types
* Created derived columns for better analysis

---

## 📊 Power BI Dashboard Features

* Interactive filters (date, transaction type)
* Total transactions KPI
* Total debit vs credit comparison
* Monthly transaction trends
* Category-wise spending analysis
